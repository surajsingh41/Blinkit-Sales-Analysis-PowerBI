# 🛒 Blinkit Grocery Sales Analysis Dashboard (Power BI)

## 📌 Overview

This project presents an interactive **Power BI dashboard** built to analyze grocery sales data from Blinkit. The dashboard provides insights into product performance, sales trends, outlet distribution, and customer purchasing behavior.

The goal is to transform raw data into meaningful insights that can support **business decisions and strategy optimization**.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing product categories
* Understand outlet-wise sales distribution
* Track trends based on outlet size and location
* Enable dynamic filtering for deeper insights

---

## 📊 Dashboard Features

### 1. Key Metrics

* Total Sales
* Average Sales
* Number of Items
* Average Rating

### 2. Product Insights

* Sales by Item Type
* Top-performing categories
* Contribution of each product segment

### 3. Outlet Analysis

* Sales by Outlet Size (Small, Medium, High)
* Sales by Outlet Location (Tier 1, Tier 2, Tier 3)
* Outlet Establishment Trends over years

### 4. Filters & Interactivity

* Outlet Location Type
* Outlet Size
* Item Type
* Custom slicers with **Clear All Filters functionality**

---

## ⚙️ How to Use

1. Download the `.pbix` file
2. Open it in **Power BI Desktop**
3. Interact with slicers to filter data
4. Use **Ctrl + Click** for multi-selection (if enabled)
5. Use the **"Clear All Filters" button** to reset dashboard state

---

## 📈 Key Insights

* Certain product categories contribute disproportionately to total revenue
* Tier 3 locations show strong sales performance in specific segments
* Medium-sized outlets often balance volume and efficiency
* Sales trends indicate growth aligned with outlet expansion
